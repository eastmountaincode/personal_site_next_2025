# Project Overview

<ul className="list-disc ml-6 pl-1">
    <li>
        A web-based live video feed application integrated into a live performance.
    </li>
    <li>
        Prompted by the performer, audience members access the site via a QR code, allowing them to project their video feed onto a projected display page.
    </li>
</ul>
